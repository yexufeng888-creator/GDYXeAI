# 网站升级计划

## 概述
对 Campus Intelligence 网站进行全面升级，增加更多液态玻璃效果、APP 即将上线公告（含效果图展示）、独立的使用指南和常见问题页面，以及图书馆二级页面。

---

## 1. 主页增强 — 液态玻璃 + APP 上线公告

### 1.1 更多液态玻璃效果（仿 Apple 官网风格）
- 在主页各 section 之间加入液态玻璃过渡卡片
- 增强导航栏的 glassmorphism 效果（更强的 blur + 彩色折射边框）
- 为 feature cards 添加液态玻璃悬浮动画（hover 时产生流体变形）
- 新增全屏液态玻璃 hero banner 背景动画（类似 Apple Vision Pro 页面的光晕流动）

### 1.2 APP 即将上线公告 Section
- 新增一个大型展示区域，标题："即将登陆 · Coming Soon"
- 展示 iPhone 效果图（使用提供的 `6fb83b10b15228caac9951539ddc61af.jpg`）
- 预留 iPad 和 Mac 效果图位置（使用占位 mockup 框架）
- 三设备并排展示，带 3D 透视倾斜效果
- 倒计时或"敬请期待"标签
- App Store / TestFlight 预注册按钮

---

## 2. 新增独立页面

### 2.1 使用指南页面 (`guide.html`)
- Apple 风格的分步教程布局
- 液态玻璃卡片展示各功能模块的使用方法
- 包含：快速开始、AI 对话、写作助手、代码调试、学习规划等模块
- 搜索功能 + 侧边导航

### 2.2 常见问题页面 (`faq.html`)
- 手风琴式 FAQ 展开/收起
- 液态玻璃风格的问答卡片
- 分类：账号相关、功能使用、隐私安全、技术支持
- 底部联系支持入口

### 2.3 图书馆页面 (`library.html`)
- **书籍查询系统**：搜索框 + 分类筛选 + 书籍卡片网格
- **借阅功能**：模拟借阅流程（选书 → 确认 → 借阅成功）
- **座位预约系统**：
  - 图书馆楼层平面图（可视化座位布局）
  - 座位状态显示（空闲/已占/已预约）
  - 时间段选择 + 预约确认
- 整体风格与主站一致，液态玻璃 UI

## 5. 技术实现要点
- 所有页面纯 HTML/CSS/JS，无需构建工具
- 液态玻璃效果使用 `backdrop-filter: blur()` + 动态渐变 + SVG filter
- APP 展示区使用 CSS 3D transform 模拟设备倾斜
- 图书馆座位系统使用纯前端 JS 模拟（localStorage 存储状态）
- 保持现有的暗色主题 + Apple 风格设计语言
