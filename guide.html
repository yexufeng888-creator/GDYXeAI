User Guide

使用指南

从零开始，快速掌握 Campus Intelligence 的全部能力。跟随以下步骤，解锁你的 AI 校园助手。

🔍
Quick Start

快速开始四步上手

无需复杂配置，几分钟即可开始使用 AI 助手提升你的学习效率。

01
注册账号
使用学号或校园邮箱注册，支持校园卡、微信一键登录。验证身份后即可使用全部功能。
🎓
02
开始对话
点击右下角 AI 助手按钮，输入你的问题。支持学术问答、代码调试、写作辅助等多种场景。
💬
03
探索工具
写作助手、创意设计、学习规划——在功能面板中选择专属工具，获得更精准的 AI 辅助。
🛠️
04
个性化设置
在设置中配置你的专业方向、学习偏好与通知频率，让 AI 更懂你的需求。
⚙️
1

AI 对话 · 智能问答

Campus Intelligence 由 ChatGPT 驱动，支持多轮上下文对话。你可以：

• 提出学术问题，获得详细解答
• 上传代码片段，获取调试建议
• 讨论设计方案，获得创意灵感
• 规划学习路径，制定复习计划

提示：问题越具体，AI 的回答越精准。尝试提供背景信息和具体需求。
You: 帮我解释一下 Python 中装饰器的工作原理

AI: 装饰器本质上是一个高阶函数，它接收一个函数作为参数，返回一个新的函数...

You: 能给我一个实际应用的例子吗？

AI: 当然！比如一个计时装饰器...
2

写作助手 · 学术增强

专为学术写作设计的 AI 工具，帮助你：

• 论文润色 — 优化表达、修正语法、提升学术性
• 摘要生成 — 一键生成符合规范的中英文摘要
• 引用格式 — 自动转换 APA / MLA / GB/T 7714
• 查重预检 — 提交前检测相似度，提升原创性
• 大纲规划 — 根据主题生成论文结构建议

在编辑器中选中文本，点击 ✨ 按钮即可调用 AI 增强。
[原文]
这个研究很重要因为它解决了一个问题

[AI 优化]
本研究具有重要的理论与实践意义，其核心贡献在于系统性地解决了...
3

代码助手 · 编程伙伴

支持 Python、JavaScript、Java、C++ 等主流语言：

• 粘贴代码，AI 自动识别语言并分析
• 逐行解释复杂算法逻辑
• 发现潜在 bug 并提供修复方案
• 推荐最佳实践与性能优化建议
• 生成单元测试代码

VS Code 集成：安装 Campus Intelligence 插件，在编辑器内直接调用 AI。
def fibonacci(n):
  if n <= 1: return n
  return fibonacci(n-1) + fibonacci(n-2)

💡 AI 建议：使用记忆化优化递归，避免指数级时间复杂度
4

学习规划 · 智能日程

与教务系统联动，AI 为你量身定制学习计划：

• 课程同步 — 自动导入课程表与考试安排
• 智能提醒 — 作业截止、考试倒计时推送
• 复习规划 — 基于艾宾浩斯遗忘曲线安排复习
• 薄弱分析 — 识别知识盲区，推荐学习资源
• 时间管理 — 番茄钟 + 专注统计

在设置中绑定教务账号，即可自动同步所有课程信息。

Pro Tips

高效使用小技巧

🎯
提问要具体
越具体的问题，AI 回答越精准。提供背景、约束条件和期望格式。
🔄
多轮追问
AI 支持上下文记忆，可以在同一对话中不断深入追问。
📎
善用附件
支持上传图片、文档、代码文件，AI 可以直接分析内容。
⌨️
快捷键
Ctrl+K 快速唤起 AI，Ctrl+Shift+A 切换工具面板。
📚
收藏对话
点击星标收藏有价值的对话，方便日后复习查阅。
🌙
深色模式
默认深色主题保护视力，也可在设置中切换浅色模式。
